\# 🖥️ Linux System Monitoring Automation



\## 📌 Overview

This project automates system monitoring on a Linux server using Bash scripting and cron jobs. It logs CPU, memory, and disk usage at regular intervals.



\---



\## 🛠️ Technologies Used

\- Linux (Ubuntu)

\- Bash Scripting

\- Cron Jobs



\---



\## ⚙️ Features

\- Monitors CPU usage

\- Tracks memory utilization

\- Checks disk usage

\- Logs system stats with timestamps
